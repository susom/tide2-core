# Sample Data

All data in this directory is **entirely synthetic and fabricated**. No real patient data is included.

The sample clinical notes were generated for demonstration and testing purposes only. Any names, dates, medical record numbers, or other identifiers are fictional. Any resemblance to real persons is coincidental.

Institution names and addresses that appear in sample notes (e.g., hospital names, street addresses) are used for realism in testing the de-identification pipeline and do not indicate any association with real patient records.
